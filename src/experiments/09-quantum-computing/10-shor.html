💡 쇼어 알고리즘의 핵심은 주기 찾기입니다. f(x) = ax mod N 은 주기 r 로 반복되며, 양자 푸리에 변환(QFT)은 이 주기 정보를 확률 봉우리로 바꾸어 한 번의 측정으로 r 을 알아내게 해줍니다. r 을 알면 gcd(ar/2 ± 1, N) 로 N 의 인수를 고전적으로 계산할 수 있습니다.

쇼어 알고리즘의 원리

문제: 큰 수의 소인수분해

두 소수의 곱 N = p × q 가 주어졌을 때 p 와 q 를 찾는 문제는, N 이 커질수록 고전 컴퓨터로는 현실적인 시간 안에 풀 수 없다고 여겨집니다. 이 어려움이 바로 RSA 공개키 암호의 안전성 근거입니다. 쇼어 알고리즘은 양자 컴퓨터로 이 문제를 다항 시간에 풀어, RSA를 위협합니다.

핵심 통찰: 인수분해 → 주기 찾기

N 과 서로소인 밑 a 를 하나 고르면, 함수 f(x) = ax mod N 은 어떤 주기 r 로 반복됩니다. 즉 ar ≡ 1 (mod N) 이 되는 가장 작은 r 이 존재합니다.

  • 1) 함수 만들기: x = 0, 1, 2, … 에 대해 ax mod N 을 나열하면 값이 주기적으로 반복됩니다.
  • 2) 양자 주기 찾기: 이 주기적 함수에 QFT를 적용해 측정하면, 측정값 k 는 Q/r 의 배수 근처에서 뾰족한 확률 봉우리를 만듭니다 (Q = 2t ≥ N²).
  • 3) 주기 추출: 측정한 k/Q 를 연분수(continued fraction)로 전개해 분모에서 r 을 복원합니다.
  • 4) 인수 계산: r 이 짝수이고 ar/2 ≢ −1 (mod N) 이면, gcd(ar/2 − 1, N) 과 gcd(ar/2 + 1, N) 가 N 의 진짜 인수입니다.

고전 컴퓨터로 주기 r 을 찾는 것은 인수분해만큼이나 어렵지만, QFT는 중첩된 모든 x 를 한꺼번에 변환해 주기를 확률 봉우리로 드러냅니다. 이것이 양자 가속의 원천입니다.

f(x) = ax mod N, 주기 r: ar ≡ 1 (mod N)
QFT 측정 봉우리: k ≈ round(m·Q/r), m = 0 … r−1
연분수: k/Q ≈ m/r → 분모 r 복원
r 짝수 & ar/2 ≢ −1 → factor = gcd(ar/2 ± 1, N)
예: N=15, a=7 → r=4 → gcd(48, 15)=3, gcd(50, 15)=5

흔한 오해

"양자 컴퓨터가 모든 경우를 동시에 시도해 답을 즉시 찾는다"는 틀린 그림입니다. 측정하면 하나의 값만 나옵니다. 쇼어의 힘은 간섭에 있습니다. QFT가 주기와 맞는 측정값의 확률만 보강 간섭으로 키우고 나머지는 상쇄시켜, 유용한 정보(주기)만 뽑아냅니다. 또한 r 이 홀수이거나 ar/2 ≡ −1 이면 실패하므로, 다른 a 로 재시도가 필요합니다.

역사적 맥락

주요 연표

  • 1977년 — 리베스트·샤미르·아델만이 소인수분해의 어려움에 기반한 RSA 공개키 암호를 발표.
  • 1994년 — 피터 쇼어(Peter Shor)가 AT&T 벨 연구소에서 소인수분해와 이산로그를 다항 시간에 푸는 양자 알고리즘을 발표. 양자 컴퓨팅 연구가 폭발적으로 시작됨.
  • 1996년 — 로브 그로버가 양자 검색 알고리즘을 발표하며 쇼어와 함께 양자 알고리즘의 두 기둥을 이룸.
  • 2001년 — IBM과 스탠퍼드 연구진이 7큐비트 NMR 양자 컴퓨터로 15 = 3 × 5 를 쇼어 알고리즘으로 최초 시연.
  • 2012년 이후 — 초전도·이온트랩 큐비트로 더 큰 수의 시연이 이어지지만, 오류 정정 없이 실용적 크기의 RSA를 깨기에는 아직 큐비트 수와 품질이 부족함.

왜 중요한가

쇼어 알고리즘은 "충분히 큰 양자 컴퓨터가 있다면 오늘날의 RSA·타원곡선 암호가 무너진다"는 것을 보여주었습니다. 이 위협 때문에 전 세계가 양자 컴퓨터로도 안전한 양자내성암호(Post-Quantum Cryptography)로의 전환을 서두르고 있습니다. 하나의 알고리즘이 학문·산업·국가 안보의 방향을 바꾼 대표 사례입니다.

피터 쇼어 (Peter Williston Shor, 1959~)

미국의 응용수학자로, 1994년 벨 연구소 재직 중 소인수분해를 위한 양자 알고리즘을 고안했습니다. 그는 주기 찾기라는 정수론 문제를 양자 푸리에 변환으로 연결해, 지수적으로 느린 고전 계산을 다항 시간으로 바꿀 수 있음을 보였습니다. 이후 쇼어는 양자 오류 정정 부호도 처음 제안해, 실제 양자 컴퓨터를 만들 수 있다는 희망의 이론적 토대를 놓았습니다. 이 공로로 괴델상, 네반린나상 등을 수상했습니다.
