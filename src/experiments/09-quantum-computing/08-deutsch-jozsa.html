도이치–조자 알고리즘

Deutsch–Jozsa Algorithm
1992 · 도이치 · 조자 (Deutsch · Jozsa) · 난이도 ⭐⭐⭐
선택한 오라클: 상수 f=0
실제 유형: 상수 (Constant)
현재 단계: ① 초기화 |00⟩|1⟩
오라클 질의 횟수: 1 회
P(입력 = |00⟩): —
측정 판정: 아직 측정 전
💡 오라클이 상수이면 마지막에 입력 레지스터를 측정했을 때 반드시 |00⟩이 나오고, 균형이면 절대 |00⟩이 나오지 않습니다. 이 판정을 위해 양자 컴퓨터는 오라클을 단 한 번만 부르면 됩니다. (고전적으로는 최악의 경우 2ⁿ⁻¹+1 = 3번 확인해야 합니다.)

도이치–조자 알고리즘의 원리

문제: 상수인가, 균형인가?

함수 f : {0,1}ⁿ → {0,1} 이 다음 둘 중 하나임을 보장(약속)받았다고 합시다.

  • 상수(Constant): 모든 입력에 대해 같은 값(전부 0 또는 전부 1)을 출력.
  • 균형(Balanced): 입력의 정확히 절반에 0을, 나머지 절반에 1을 출력.

목표는 f가 상수인지 균형인지 판별하는 것입니다. 고전 컴퓨터는 최악의 경우 입력 2ⁿ개 중 절반보다 한 개 더 많은 2ⁿ⁻¹ + 1번을 확인해야 확신할 수 있습니다. (n=2이면 3번.) 그러나 양자 알고리즘은 오라클을 단 한 번 불러 답을 냅니다.

위상 킥백 (Phase Kickback)

핵심은 보조 큐비트(ancilla)를 |−⟩ = (|0⟩ − |1⟩)/√2 상태로 준비하는 것입니다. 오라클 U_f: |x⟩|y⟩ → |x⟩|y⊕f(x)⟩ 을 |−⟩에 적용하면, f(x)=1일 때 |0⟩과 |1⟩이 뒤바뀌어 전체 부호가 반전됩니다. 즉 U_f|x⟩|−⟩ = (−1)^{f(x)} |x⟩|−⟩ 이 되어, 함수 값이 입력 레지스터의 위상으로 "튕겨" 들어옵니다.

입력에 하다마드(H)를 걸어 만든 균등 중첩에 이 위상을 실은 뒤 다시 H를 걸면, 간섭에 의해 상수일 때는 모든 위상이 같아 |00…0⟩에 완전히 보강 간섭되고, 균형일 때는 |00…0⟩에서 정확히 상쇄됩니다.

초기: |0⟩⊗ⁿ |1⟩
H⊗ⁿ⁺¹ 후: (1/√2ⁿ) Σₓ |x⟩ ⊗ |−⟩
오라클(위상 킥백): (1/√2ⁿ) Σₓ (−1)^{f(x)} |x⟩ ⊗ |−⟩
H⊗ⁿ 후 |0…0⟩의 진폭 = (1/2ⁿ) Σₓ (−1)^{f(x)}
상수 → |진폭| = 1 → P(|0…0⟩) = 100%
균형 → 진폭 = 0 → P(|0…0⟩) = 0%

흔한 오해

"양자 컴퓨터가 모든 입력을 동시에 계산해서 하나하나 읽어낸다"는 것은 틀린 그림입니다. 중첩 상태를 측정하면 값 하나만 무작위로 얻을 뿐이라, 2ⁿ개의 f(x)를 전부 읽을 수는 없습니다. 도이치–조자의 힘은 개별 값이 아니라 f 전체의 전역적 성질(상수/균형)을 간섭으로 한 번에 뽑아낸다는 데 있습니다.

퀴즈: 확인해 봅시다

n=2 (입력 4가지)일 때, 고전 컴퓨터가 상수/균형을 확실히 판별하려면 최악의 경우 몇 번 f를 계산해야 할까요?

역사적 맥락

주요 연표

  • 1985년 — 데이비드 도이치(David Deutsch)가 보편 양자 튜링 기계를 정식화하고, 단일 비트 함수에 대한 "도이치 문제"를 제시(1큐비트 판별).
  • 1992년 — 도이치와 리처드 조자(Richard Jozsa)가 이를 n비트로 확장한 도이치–조자 알고리즘을 발표. 양자가 고전보다 확정적으로 지수적으로 빠른 최초의 명확한 사례.
  • 1994년 — 피터 쇼어의 소인수분해 알고리즘 발표로 양자 알고리즘 연구가 폭발적으로 확대됨.
  • 1998년 — 핵자기공명(NMR) 기반 소규모 양자 시스템에서 도이치–조자 알고리즘이 실험적으로 시연됨.

왜 중요한가

도이치–조자 알고리즘은 실용적 응용은 크지 않지만, 양자 병렬성과 간섭·위상 킥백이라는 양자 알고리즘의 핵심 도구를 가장 깔끔하게 보여주는 "교과서 예제"입니다. 이후의 번스타인–바지라니, 사이먼, 쇼어 알고리즘이 모두 여기서 발전한 아이디어를 공유하며, "양자가 고전을 확실히 앞선다"는 개념적 출발점이 되었습니다.

데이비드 도이치 (David Deutsch, 1953~) · 리처드 조자 (Richard Jozsa, 1953~)

도이치는 이스라엘 태생의 영국 물리학자로 옥스퍼드에서 활동하며 "양자 컴퓨팅의 아버지"로 불립니다. 그는 보편 양자 컴퓨터의 개념을 정립했습니다. 조자는 오스트레일리아 출신의 수학자·물리학자로 양자 정보 이론, 특히 얽힘과 양자 알고리즘 분야에 크게 기여했습니다. 두 사람의 1992년 공동 연구는 양자 알고리즘 이론의 초석이 되었습니다.
