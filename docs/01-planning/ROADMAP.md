- **학습 경로 / 가이드 투어** — 초급→고급 코스, 교실 플레이리스트 연동.

- **물리 회귀 테스트 확대** — 현재 대표 4종(블로흐·그로버·벨·도이치–조자). 전 시뮬레이션에 자기검증 훅(`window.__PHYSICS_CHECK`)을 심어 커버리지 확대.
- **시각 회귀 테스트**(스크린샷 diff) + **Lighthouse 성능 예산**을 CI에 추가.

## 3. 진행 추적 (Progress Tracking)

- 진행 상태는 `src/data/experiments.json`의 `status` 필드로 관리한다: `planned` → `in-progress` → `ready`. **현재 70개 전부 `ready`.**
- 허브 `src/index.html`는 `status`를 읽어 카드 표시를 제어한다(`ready`만 링크 활성화).
- 실험 페이지 경로 규칙: `src/experiments/{category}/{slug}.html` (허브 링크는 `./experiments/{category}/{slug}.html`).
- 새 실험은 (1) `experiments.json`에 메타·아이콘·링크 등록 → (2) 3탭 자체 완결 HTML 작성(Zero Dependencies) → (3) 생성기로 공유 크롬(상단바·i18n·교실모드·관련/더알아보기) 주입 → (4) QA·접근성·물리 회귀 테스트 통과 → (5) 배포.
- **완료 정의(DoD):** 3탭(🎮 인터랙티브 / 📚 원리 학습 / 📜 역사적 맥락) 완비, 단일 HTML 독립 실행, Zero Dependencies, 콘솔 에러 0, WCAG AA 무위반, Chromium·Firefox·WebKit 동작 확인, 반응형.

## 부록 A. 초기 v1.0 개발 계획 (아카이브)

아래는 최초 53개 실험을 10주(6 Phase)에 구현한 원래 계획으로, 역사적 참고용으로 보존한다.

| Phase | 기간 | 목표 |
|-------|------|------|
| 1 | Week 1 | 문서화, 허브 페이지, 파일럿 3개 실험 |
| 2 | Week 2-3 | 01-ancient (4), 02-scientific-revolution (7) |
| 3 | Week 4-5 | 03-precision-era (5), 04-energy-field (7) |
| 4 | Week 6-7 | 05-atomic (9), 06-quantum (7) |
| 5 | Week 8-9 | 07-relativity (7), 08-modern (7) |
| 6 | Week 10 | QA, 문서 최종화, v1.0 릴리스 |

실험 우선순위는 **시각적 임팩트 × 직관성 × 구현 난이도**로 P1/P2/P3 세 등급으로 분류해, 각 Phase 안에서 P1 → P2 → P3 순으로 착수했다. (원 계획 기준 P1 20개 / P2 22개 / P3 11개, 총 53개.)
